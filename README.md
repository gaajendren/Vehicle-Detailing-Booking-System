## Project Description

The **Vehicle Detailing Booking System** is a web-based system designed to manage vehicle detailing bookings and streamline the service process. The system provides different access levels for **Customers**, **Staff**, and **Administrators** to handle booking activities according to their roles.

### Customer
Customers can:
- Register and log in to the system
- View available detailing services
- Create vehicle detailing bookings
- Manage their bookings
- View booking status and history

### Staff
Staff can:
- Log in to the system
- View customer bookings
- Manage booking status
- Handle detailing service requests

### Admin
Administrators can:
- Manage users (customers and staff)
- Manage detailing services
- View and manage all bookings
- Monitor overall system activities

The system helps organize vehicle detailing appointments and improves the efficiency of managing customer requests and service operations.
